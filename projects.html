Корпоративный портал АЛЕАН-Турне

Два корпоративных портала Bitrix24 для туристических агенств и частников. Разработка архитектуры по обмену данными между Soap сервисами и внешним сайтом alean.ru. Написание функционала для менеджеров в CRM Bitrix24, настройка воронок и автоматическое создание отчетов. Добавление событий по отправке SMS и E-mail сообщений с сохранением в таймлайне на детальной странице сделки. Формирование отчетов с использованием модуля documentgenerator. Функционал был реализован с использованием D7, код стандартизирован в рамках Bitrix'a

2
Интернет магзин BMW Group Россия

Оффициальный интернет магазин BMW Group в России по продаже аксессуаров. Разработка архитектуры по реализации RESTApi между фронт & бэк частью сайта в рамках Битрикса. Написание собственного модуля по реализации REST, с возможностью конфигурирования через Административную часть сайта. Добавление информационных блоков/highloadblock. Разработка контроллеров для Новостей, статичных страниц, каталога, корзины, оформлении заказа, меню и фильтров. Написание собственного компонента для оформления заказа на сайте с использованием D7. Разработка собственного умного фильтра в рамках битрикса с возможностью задавать ограничения по фильтрации полей через Административную чать. Работа с HTTP заголовками, реализация ответов 200, 403, 400, 404 и 500 для контроллера REST. Фильтрация IP адресов при работе с RESTApi. Обратная интеграция с BMW CRM и AmoCRM по RESTApi. Добавление контроллеров для работы с вебхуками.

3
Личный кабинет сотрудника Холдинга Таграс

Разработка архитектуры для личного кабинета сотрудника холдинга Таграс с возможностью заказа копий документов, справок, ознакомлением с регламентами и информированием. Личный кабинет поддерживает функционал по оформлению 11 типов заявлений и 15 видами заявок. Система фиксирует время исполнения заявок, подсчитывает среднюю оценку качества обслуживания как в целом, так и по каждому специалисту кадровой службы. Написание функционала для Заявок, Заявлений, Справочников, Личных документов и отчетов. Формирование отчетов для руководителей отделов, с возможностью фильтровать по дате и сотруднику/организации. Разработка собственных комлпексных компонентов в рамках битрикса, с поддержкой "Эрмитаж"

4
Веб-портал конкурсных закупок УРПС Татнефть

Тендерный портал УРПС ПАО «Татнефть» предназначен для проведения открытых и закрытых торгов по продуктам и услугам зарегистрированных организаций. Доработка и модернизация существующего функционала на сайте. Реализован механизм автоматического перерасчета позиций лота и изменение условия отправки предложения участником торгов. Написание "Агента", с возможностью рассылки E-mail сообщений пользователям о завершении редукциона. Добавление почтовых событий и шаблонов.
